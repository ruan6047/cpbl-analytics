## 背景

現行能力卡守備軸 `RF = (PO+A)/G` 衡量「經手多少球」而非守備價值；改分母只修正時間基準，
不改變衡量對象（見 [`UX-ABILITY-FIELD1_PHASE1.md`](../research/UX-ABILITY-FIELD1_PHASE1.md)，
該卡 Phase 2 已依需求方裁決不執行）。

國外對「無座標資料」的處境有成熟解法：[Total Zone](https://www.baseball-reference.com/about/total_zone.shtml)
只需逐打席的「誰處理的」，估算超越期望的出局數。我們的 `game_livelog` 2018–2026 有約
**13.6 萬筆**可抽出守位的事件（逐球追蹤僅 5,998 筆），且可跨季累積以滿足
[UZR 文件](https://library.fangraphs.com/defense/uzr/)明訂的「三年才可下結論」門檻。

## 研究問題（本卡要回答的）

- [x] **文字解析涵蓋率**：**95.31%**（138,370／145,178 擊球入場），九季穩定 94.88–96.26% 無斷層。殘餘 4.69% **有系統性偏誤**：犧牲觸擊 2,159 筆全數落空（「擊出球(觸擊)」無方向詞），系統性低估投／捕／一／三壘手機會。詳 [報告](../research/ML_FIELD_TZ1_FEASIBILITY.md) §1。
- [x] **期望值模型怎麼建**：**naive 查表法否決、迴歸式可行**（iteration 2 依查核 Finding 1 更正）。`content` 的「方向」是處理結果非落點（內野滾地球出局率 89.3% vs 滾出外野 0.2–0.5%），故 naive 查表法屬循環論證＋單顆球歸屬欠定（失敗僅 10–17／5,000 筆具名）。但「單顆欠定」≠「母體不可識別」：九季配對變異下，混合效應 logistic 迴歸對 SS／3B 個人參數可識別。詳 §2。
- [x] **跨季累積與年代校正**：**須分類別處理**（iteration 2 依查核 Finding 2 更正）。高飛球出局率九季平坦（68.11%→68.98%）可直接合併；但**平飛球有嚴重判定漂移**（11.45%→1.95%，接殺平飛球被改記高飛球），跨季合併須以年份 fixed effect 吸收，否則低估早期外野守備。詳 §3。
- [x] **驗證方案**：已執行場次數對帳（逐年與 spec 基線 §3 完全一致）與每場打席數對照官方 PA（+1.37% 過度切分不污染出局率，查核 Finding 3 佐證）。若實作建議三階：出局守恆 ±3%、與官方守備率中度正相關、金手套僅作弱訊號不可當 ground truth。詳 §4。
- [x] **規模與風險評估**：發現未預見前置——**無逐場守備陣容**（`game_detail` 無先發、`fielding_innings` 僅球季層級），須由打者列 `defend_station_code` ＋換防事件重建（迴歸式路線的關鍵前置，不需新爬蟲）。外野三季累積約 885 次機會、精度 ±14 個機會。核心風險「看似精確實則錯誤」限於 naive 查表法；迴歸式可用配對變異＋年份 fixed effect 控制。詳 §5。

## 研究結論（執行者，iteration 2）

- **naive 期望值查表法 TZ：NO-GO** —— 無座標下查表等於用結果預測結果（循環論證＋單顆球歸屬欠定），無有效控制手段。
- **迴歸式 TZ（球員配對混合效應）：條件式 GO** —— collapse 物理方向破循環＋逐局陣容重建提供 SS/3B 變數＋年份 fixed effect 校正平飛球漂移；工程量 L、三年門檻、精度 ±14 機會。屬另卡並須需求方核可。
- **短期內仍不建議以 TZ 取代能力卡守備軸**：管線成熟前「外野有值、內野沿用 RF」跨守位不可比；取代守備軸應待迴歸式路線驗證後再議。零風險緩解仍是 spec 基線方向三（改標示／多季 rolling）。
- 四條可行路徑（迴歸式 TZ／僅外野 OAA／DRA／多季標示）詳報告 §6.1。
